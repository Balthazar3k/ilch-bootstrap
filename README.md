ilch meets bootstrap
==============

Ilch Clan CMS 1.1p Frontend an Bootstrap angepasst

Es können somit Templates auf Bootstrap Basis erstellt werden und mit dieser Erweiterung in Verbindung mit dem Ilch 1.1p Frontend genutzt werden.
Optimale Nutzung unter den neuen Endgeräten, wie PC, Tablet oder Smartphone.

Mit im Packet befinden sich 2 Templates,sowie eine modifizierte Loginbox.
Die Templates dienen nur zur Ansicht der Ilch 1.1p Bootstrap Version. Es werden keine weiteren Anpassungen der Templates über den Support vorgenommen!

Readme zum Template siehe Ordner ( Dateien öffnen mit Browser)



Zur Info

Bootstrap ist ein Framework mit HTML, CSS und JS .Erstelle dir mit Hilfe der Anleitungen auf Bootstrap oder externer Seiten dein eigenes Layout,nach deinen Vorstellungen und nutze es in Verbindung mit dem Ilch CMS.
