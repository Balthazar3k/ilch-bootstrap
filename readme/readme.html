Ilch meets Bootsrap

Voraussetzung

Es sollte die Ilch Clan version 1.1p installiert sein, sowie BBcode 2.0 für Ilch 1.1p.
Erst im Anschluss diese Erweiterung aufladen und bestehende Dateien ersetzen.Das überspielen dieser Erweiterung auf eine bestehende Version mit ggf. installierten Modulen kann zu Fehlern führen.
Eine Ilch 1.1p Version nur mit bbcode 2.0 wird empfohlen.
Bei aufkommenden Fehlern übernehme ich keine Haftung!Vorher ein Backup der Seite erstellen!

Installation

Alle Dateien aus dem Ordner upload nach include aufladen und bestehende Dateien ersetzen.
Ggf. Backup der Seite vorher erstellen.

Eigenes Design erstellen

Mit Hilfe der Anleitungen auf Bootstrap kannst du dir ein individuelles Design erstellen.
Die benötigten Dateien sind bereits vorhanden und müssen nur in der index.htm eingetragen werden:

<link href="include/bootstrap/ekko-lightbox.css" rel="stylesheet">
<link href="include/bootstrap/font-awesome/css/font-awesome.min.css" rel="stylesheet">
<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.1/jquery.min.js"></script>
<script src="include/bootstrap/js/bootstrap.js"></script>
<script src="include/bootstrap/js/bootstrap.file-input.js"></script>
<script src="include/bootstrap/js/ekko-lightbox.js"></script>
<script src="include/bootstrap/js/bootstrap-popover.js"></script>
<script src="include/bootstrap/js/bootstrap-tooltip.js"></script>
markieren
Auch befindet sich die bootstrap.css bereits an Board.Du kannst diese in die index.htm vom Design eintragen und die Änderungen über die style.css definieren oder dir eine Eigene in den Designordner legen.

<link href="include/bootstrap/bootstrap.css" rel="stylesheet">
markieren
Bei Fragen kontaktiere einfach den Supportbereich auf Maretz.eu.

Auch kannst du unter Anfrage auf Maretz.eu ein Modul zur Anpassung vorschlagen.
